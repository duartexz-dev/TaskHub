# 🚀 TaskHub

**TaskHub** é uma aplicação web simples criada para organizar tarefas e praticar conceitos de desenvolvimento **front-end**.
O projeto foi desenvolvido utilizando **HTML, CSS e JavaScript**, focando em manipulação do DOM, organização de tarefas e interação do usuário com a interface.

---

## 📌 Sobre o Projeto

O **TaskHub** funciona como um pequeno gerenciador de tarefas (to-do list).
Ele permite que o usuário insira informações e organize tarefas de forma simples diretamente no navegador.

Este projeto também foi criado com o objetivo de **treinar lógica de programação e desenvolvimento web**.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** — Estrutura da aplicação
* **CSS3** — Estilização da interface
* **JavaScript** — Lógica e interatividade
* **LocalStorage** — Armazenamento de dados no navegador
* **Bootstrap** — Componentes e layout responsivo

---

## ⚙️ Funcionalidades

✔ Cadastro de nome e idade
✔ Validação de dados do usuário
✔ Criação de tarefas
✔ Lista de tarefas dinâmica
✔ Interface simples e organizada
✔ Armazenamento de dados no navegador
